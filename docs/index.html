ShigotoForm

あなたの履歴書は、あなたのデバイスの中に。
サーバー送信ゼロで作れる、日本式履歴書作成PWA。

🚀 今すぐ使ってみる GitHub で見る

アカウント登録不要・完全無料・インストール不要（PWA対応）

履歴書ツールの「不安」を、ぜんぶ解決

氏名・住所・職歴——履歴書は個人情報のかたまりです。ShigotoForm はデータを一切送信しない設計で、その不安を根本から取り除きます。

🔒

個人情報がクラウドにアップロードされる

完全クライアントサイド。データはブラウザの IndexedDB のみに保存され、外部への通信は一切ありません。

🗾

海外製ツールは日本式履歴書に非対応

履歴書レイアウトを、ゴシック体・明朝体の日本語フォントで表現します。顔写真添付には近日対応予定。

💸

PDF出力は有料プラン限定

html2pdf.js によるローカル生成で、高解像度の A4 PDF を無料でダウンロードできます。

💾

機種変更でデータが消える

JSON エクスポート／インポートでバックアップと復元が自由自在。デバイスをまたいで引き継げます。

📶

ネット環境がないと使えない

Service Worker によるオフライン対応。PWA としてインストールすれば圏外でも編集できます。

♿

アクセシビリティが後回し

Lighthouse アクセシビリティスコア 100、WAVE 検証エラー 0 件。誰にでも使いやすい UI です。

使い方は、たったの 3ステップ

面倒な設定は不要。開いたらすぐ書き始められます。入力内容は自動保存されるので、途中で閉じても大丈夫。

✏️ 入力する

氏名・住所・学歴・職歴・資格をフォームに入力。ふりがなの自動補完や行の追加・削除にも対応しています。

👀 プレビューする

A4 用紙サイズで仕上がりを確認。ゴシック体・明朝体をワンクリックで切り替えられます。

📥 PDFで保存する

「履歴書PDFダウンロード」を押すだけ。印刷にそのまま使える高解像度 PDF が手に入ります。

スクリーンショット

シンプルな入力画面と、印刷品質のプレビュー。ダークモードにも対応しています。

信頼できる技術スタック

モダンで実績のあるオープンソース技術のみで構成。CI では依存パッケージのマルウェアスキャン（Takumi Guard）も実施しています。

  • TypeScript
  • Vite
  • PWA / Service Worker
  • Dexie (IndexedDB)
  • html2pdf.js
  • Bootstrap
  • Biome
  • Playwright E2E
